## ğŸ’» Getting Started <a name="getting-started"></a>

- To get a local copy up and running, follow these steps.

### Prerequisites

- A code editor of your choice (like vs code or atom and so on)
-version control system (git is preferred)

### Setup

- Create a new folder to store the files in.

### Install

Install this project with:

- Open your command prompt or terminal and run.
- git clone https://github.com/Kanyorok/CodeFrontUsers.git
- cd CodeFrontUsers
- install xampp or alternative local host or server.
- start server (important Apache and Mysql).
- visit phpmyadmin page http://localhost/phpmyadmin/.
- import the database codefront_crud.sql
- launch app in browser by navigating http://localhost/CodeFrontUsers/index.php

### Usage

- this is a developer management app and it is used to manage personal data about developers.
